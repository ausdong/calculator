Our app is a simple calculator.

To use this app, simply push the buttons!

Operations with 2 explicitly inputted numbers and an operation will work correctly e.g. 4+2, 832*13, 1039/18444, 1+1
Operating on a result works as well, e.g. 4+2 = 6, +5 = 11
Other functions are not working or are limited on performance.
