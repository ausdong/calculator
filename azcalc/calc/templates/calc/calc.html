Calculator

AD & ZZ Calculator App

The upper text field shows input.
The lower text field shows result.
'Clear' will reset everything except whatever is in memory 'm'.
'm+' attempts to store a number from the input box.
'm-' will clear the memory 'm'.
'm' will recall the stored number into the text box.

*Note: memory functions and anything beyond adding/subtracting/multiplying/dividing two explicit numbers are not functioning as intended
